# University Result System (Java)

A Java-based application that manages student results, calculates percentages, CGPA, grades, and generates merit lists.

## Features

* Student Result Management
* Percentage Calculation
* CGPA Calculation
* Grade Generation
* Merit List Generation
* Topper Identification
* Exception Handling
* Java Streams API

## Technologies Used

* Core Java
* Collections Framework
* Java Streams API
* Exception Handling
* Java Time API

## Concepts Demonstrated

* Object-Oriented Programming (OOP)
* Collections
* Streams & Lambda Expressions
* Exception Handling
* Sorting and Ranking Algorithms
